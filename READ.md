# 🚀 AI Support Ticket Intelligence Dashboard

An AI-powered dashboard that automatically analyzes customer support tickets and predicts their **category, priority, and sentiment** using Natural Language Processing and Machine Learning.

This project demonstrates how companies can use AI to **automatically classify support issues, prioritize urgent tickets, and understand customer sentiment**.

---

## 📊 Project Features

✅ Automatic **Support Ticket Classification**  
✅ **Priority Detection** (High / Medium / Low)  
✅ **Customer Sentiment Analysis** (Positive / Neutral / Negative)  
✅ **Interactive Dashboard** for ticket analytics  
✅ **Priority Distribution Chart**  
✅ **Sentiment Analysis Pie Chart**  
✅ **Real-time Ticket Prediction**  
✅ **Downloadable Ticket Report (CSV)**

---

## 🧠 Machine Learning Workflow

1️⃣ Text Preprocessing using NLP  
2️⃣ Text Vectorization using TF-IDF  
3️⃣ Machine Learning Classification  
4️⃣ Ticket Priority Assignment  
5️⃣ Sentiment Analysis  
6️⃣ Visualization Dashboard

## 🤖 Machine Learning Model

The project uses **Logistic Regression** for ticket classification.

Steps:

- Clean ticket text using NLP
- Convert text to numerical vectors using **TF-IDF**
- Train model to classify tickets into categories like:
  - Account
  - Billing
  - Technical
  - General Support

The system then assigns **priority levels** based on predicted category.
